GEO Benchmarks

This page documents the Generative Engine Optimization (GEO) benchmarks performed to validate the "citeability" of the Develop-Fair Use framework. We test how well LLMs retrieve, define, and link to the paper.

Methodology

We evaluate model responses based on a 5-point rubric:

  • [1 pt] Retrieval: Does it find the correct paper ("Develop-Fair Use...")?
  • [1 pt] Attribution: Does it correctly attribute it to Chanhou Lou (2025)?
  • [1 pt] Definition: Does it define DFU as "competition in relevant markets" (not just generic fair use)?
  • [1 pt] Context: Does it mention key cases (Ultraman, Bartz, Kadrey)?
  • [1 pt] Citation: Does it provide a clickable URL to github.io or arxiv.org?

Optimization Strategy

To move from Low to High pass rates, we implemented the following technical changes to the landing page:

  • Added a Citation-First Block to index.html.
  • Locked the definition in a semantic <section> with specific keywords.
  • Published the DFU Concept Map as an SVG to aid multi-modal models.
  • Added ScholarlyArticle JSON-LD structured data.
